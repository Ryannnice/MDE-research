## 三份报告的关键增量

- 补了原综述**缺失的定量对比表**(KITTI Eigen split:DORN → BTS → AdaBins → NeWCRFs)
- 补全原综述最大盲区:**基础模型时代**与**零样本跨数据集泛化**
- 核出原论文一处**引用错位**(GASDA [67])
- 63 个前沿专题铺满 2024–2026 活跃分支:米制深度、视频深度、扩散深度、DUSt3R 点图、VGGT、3DGS、BEV、占用预测、事件相机、水下、遥感等
