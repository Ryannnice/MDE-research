# I2: 面向几何世界模型的标定信念图

## 一句话

现有几何基础模型常把深度、点图或占用当作确定事实交给下游;本选题把确定性几何输出升级为可标定、可消费的 `geometry belief map`,并用稀疏视角 3D 重建先证明它能减少错误几何传播。

英文工作名:

```text
Calibrated Geometry Belief Maps for World Models
```

推荐首版收窄名:

```text
Calibrated PointMap Priors for Sparse-View 3D Reconstruction
```

## Target Venue and Assumptions

| 项 | 当前设定 |
|---|---|
| 目标 venue family | CCF-A CV/ML: CVPR/ICCV/NeurIPS/ICLR 候选;ECCV 可作为相邻视觉顶会备选 |
| 研究阶段 | `needs-search` + `needs-evidence-design`: idea 已成形, novelty 和实验闭环待核验 |
| 核心场景 | 几何基础模型输出点图/深度,下游 3DGS/NeRF/occupancy/world model 消费这些几何 |
| 首版任务 | 稀疏视角 3D 重建中的 uncertainty-weighted point/depth prior |
| 非目标 | 不做单纯“Depth Anything + 方差头”;不以刷单目 AbsRel 作为主贡献 |
| 结果纪律 | 所有实验状态保持 `待设计` / `待跑`,不写未验证提升 |

当前没有做最新 prior-art 检索,因此不能声称“首个”或“首次”。更稳妥的 novelty 写法是:把标定几何 belief 作为几何基础模型和下游世界模型之间的接口,并用下游错误传播减少来验证其价值。

## Raw Idea Diagnosis

原始想法:

```text
现有几何世界模型把点图当确定事实;我们把点图改造成标定 belief map,并证明它能减少错误几何传播。
```

这个方向值得保留,但需要三个收束:

| 问题 | 风险 | 收束方式 |
|---|---|---|
| “不确定性头”本身太常见 | reviewer 会认为只是 UQ/evidential regression 套壳 | 主贡献改为 `calibrated geometry interface + downstream consumption` |
| “世界模型”范围太大 | 一篇论文难同时覆盖 video prediction、planning、BEV、occupancy、3DGS | 首版只做 pointmap/depth prior -> sparse-view 3D reconstruction |
| 只证明标定不够 | ECE/NLL 好看但下游可能无收益 | 必须证明低置信几何被降权后,错误传播减少 |

## Problem Statement

给定观测序列:

```math
I_{\le t} = \{I_1,\ldots,I_t\}
```

几何基础模型输出确定性几何:

```math
\hat{\mathbf{G}}_t = F_\theta(I_{\le t})
```

其中 `G` 可以是深度、点图、占用、3D Gaussian 参数或 scene flow。现有下游通常把它当作确定事实:

```math
\hat{\mathbf{G}}_t \approx \mathbf{G}_t
```

但在反光、透明、远距离、动态物体、遮挡边界、低纹理和 OOD 相机场景中,这个假设会把局部错误变成全局错误。目标应改为估计一个可被下游消费的 belief state:

```math
p(\mathbf{G}_t \mid I_{\le t})
```

首版把 `G` 具体化为点图或深度诱导点图:

```math
p(\mathbf{X}(p)\mid I_{\le t})
=
\mathcal{N}
\left(
\boldsymbol{\mu}_X(p),
\boldsymbol{\Sigma}_X(p)
\right)
```

并回答一个可检验问题:

```text
当单目/少视角几何先验局部错误时,标定 belief map 是否能让下游重建系统少传播这些错误?
```

## Core Insight

确定性点图的失败不只在于:

```math
\boldsymbol{\mu}_X(p) \ne \mathbf{X}^\star(p)
```

更在于下游把这个错误当成高置信硬约束。一个有用的 belief map 必须同时满足两点:

1. `calibrated`: 预测的不确定性和真实几何残差频率一致,至少能排序出高风险区域。
2. `actionable`: 下游优化或预测真的使用这个信念,而不是只把不确定性画成热力图。

因此核心贡献不应写成“我们估计 uncertainty”,而应写成:

```text
我们把几何基础模型的确定性输出改造成可标定、可消费的 belief interface,
并验证这个 interface 能减少下游错误几何传播。
```

## Method Blueprint

### 1. 表示:从点图到 belief map

对每个像素或 token `p`,定义:

```math
\mathcal{B}(p)
=
\left\{
\boldsymbol{\mu}_X(p),
\boldsymbol{\Sigma}_X(p),
c(p),
r(p)
\right\}
```

其中:

| 符号 | 含义 |
|---|---|
| `mu_X(p)` | 点图均值,来自冻结 DUSt3R/VGGT/深度转点图 |
| `Sigma_X(p)` | 3D 几何不确定性,首版可用标量或对角协方差近似 |
| `c(p)` | 可选置信度,用于 gating 或可视化 |
| `r(p)` | 可选 failure-type proxy,如边界、动态、反光/透明、远距离 |

首版推荐用最小表示:

```math
\mathcal{B}_0(p)
=
\left\{
\boldsymbol{\mu}_X(p),
\sigma_X(p)
\right\}
```

其中 `sigma_X(p)` 是点图或深度残差尺度。这样实现简单,也方便和 depth gradient、photometric residual、ensemble variance 等 baseline 对齐。

### 2. 骨干与 belief head

冻结几何均值模型:

```math
\boldsymbol{\mu}_X(p), h(p) = F_\theta(I_{\le t})
```

训练轻量 belief head:

```math
s(p) = H_\phi(h(p), \boldsymbol{\mu}_X(p), q(p))
```

```math
\sigma_X(p) = \operatorname{softplus}(s(p)) + \epsilon
```

`q(p)` 是可选低成本 cue,例如深度梯度、图像梯度、重投影残差、运动一致性或语义边界。首版应先做两种版本:

| 版本 | 输入 | 目的 |
|---|---|---|
| Head-only | `h(p), mu_X(p)` | 测试冻结几何特征是否已有错误感知能力 |
| Head + cues | `h(p), mu_X(p), q(p)` | 测试显式几何/图像 cue 是否提升 failure slice |

### 3. 训练与标定目标

如果有 3D/深度监督,定义几何残差:

```math
e(p)
=
\rho\left(
\boldsymbol{\mu}_X(p) - \mathbf{X}^\star(p)
\right)
```

一个最小可用的 heteroscedastic 目标是:

```math
\mathcal{L}_{\text{cal}}
=
\sum_p
\left[
\frac{e(p)}{\sigma_X(p)+\epsilon}
+
\lambda \log(\sigma_X(p)+\epsilon)
\right]
```

这个目标本身不保证标定,所以必须在 held-out calibration split 上做后处理或校准检查:

```math
\Pr(e(p) \le \tau_\alpha(\sigma_X(p))) \approx \alpha
```

可选标定器包括 temperature scaling、isotonic regression、分桶校准或轻量 monotonic calibrator。若使用 evidential regression,可以输出 Normal-Inverse-Gamma 或其几何版本参数,但没有 ensemble/OOD 证据前不要强称 epistemic uncertainty,只称 `epistemic proxy` 或 `model-risk proxy`。

### 4. 下游消费接口

belief map 必须进入下游目标。以 3DGS/NeRF 几何正则为首版闭环:

```math
w(p)
=
\operatorname{clip}
\left(
\exp(-s_{\text{cal}}(p)),
w_{\min},
w_{\max}
\right)
```

```math
\mathcal{L}_{\text{geo}}
=
\sum_p
w(p)\,
\rho\left(
\hat{\mathbf{G}}(p)-\boldsymbol{\mu}_G(p)
\right)
```

其中 `s_cal(p)` 是校准后的 log-scale 或风险分数。这个设计要证明:

```text
高风险几何不是被删除,而是从硬约束变成软约束。
```

世界模型扩展版可写成 token gating:

```math
\tilde h_G(p)
=
\alpha(p) h_G(p),
\quad
\alpha(p)
=
\operatorname{sigmoid}
\left(a-b\,s_{\text{cal}}(p)\right)
```

但这应作为第二阶段,不要放进首版主实验里承诺太多。

## Recommended First Version

首版建议叫:

```text
Calibrated PointMap Prior for Sparse-View 3D Reconstruction
```

最小闭环:

| 步骤 | 决策 |
|---|---|
| 几何均值 | 使用冻结 DUSt3R/VGGT 类 pointmap;若实现成本高,先用强 metric depth 转点图 proxy |
| belief head | 训练轻量 `sigma` head,不大规模微调骨干 |
| 标定 | 独立 calibration split,报告 ECE/NLL/AUSE/coverage |
| 下游 | 3DGS 或 NeRF sparse-view reconstruction,把 `sigma` 作为几何正则权重 |
| 失败切片 | 透明/反光、遮挡边界、动态物体、远距离、低纹理、跨域相机 |
| 主结论 | 标定 belief map 是否比确定性点图、启发式置信度和未标定方差更少传播错误几何 |

这样做的好处是问题足够窄,但仍然保留世界模型叙事:点图 belief 是几何 world state 的可靠接口。

## Baselines and Ablations

### Baselines

| 类别 | Baseline | 用途 |
|---|---|---|
| 确定性几何 | 原 DUSt3R/VGGT/Depth Anything/Metric3D 输出 | 证明硬信几何会传播错误 |
| 常数不确定性 | 所有像素同一 `sigma` | 排除 loss reweighting 本身带来的偶然收益 |
| 图像/几何启发式 | depth gradient、RGB gradient、edge detector、photometric residual | 排除只是边缘/纹理检测 |
| 分布/采样代理 | bins entropy、ensemble variance、diffusion sample variance | 对比常见 uncertainty proxy |
| 未标定方差 | 只训练 variance head,不做 calibration split | 证明“标定”不是装饰 |
| Oracle 上界 | 用真实误差生成权重 | 判断下游任务理论可获益空间 |

### Ablations

| 消融 | 问题 |
|---|---|
| scalar `sigma` vs diagonal covariance | 3D 协方差是否真的必要 |
| Head-only vs Head + cues | 错误感知来自骨干特征还是显式 cue |
| calibrated vs uncalibrated | 标定是否带来下游收益 |
| mean frozen vs light mean adaptation | 收益来自 belief 还是均值变好 |
| weighting function 选择 | `exp(-s)`、inverse-variance、threshold gating 哪个稳定 |
| post-hoc pruning vs training-time weighting | 证明采样/优化期消费 belief 比后处理更有效 |

## Relation to I1

I1 和 I2 解决两个不同问题:

```text
I1: 如何把扩散深度骨干的采样轨迹快速米制化?
I2: 如何让下游知道几何输出该信多少?
```

可组合路线:

```text
I1 输出 near-single-step metric depth / point proxy
I2 为该 depth / pointmap 预测 calibrated belief
下游 3D 重建或世界模型按 belief 消费几何
```

但论文推进上建议分开:

| 选题 | 主贡献 | 首要证据 |
|---|---|---|
| I1 | 扩散采样期的 metric correction mechanism | 少步采样中 metric 修正传播到无锚区域 |
| I2 | 几何 belief interface + 下游可靠消费 | 标定 belief 减少错误几何传播 |
