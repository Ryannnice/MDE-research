# A2: 免训练 1-2 步扩散米制深度

## 一句话

冻结扩散深度骨干,用 Tweedie 代理锚构造 proxy-anchored CCF,并把单目几何锚烘焙进 1-2 步采样轨迹。核心问题是:采样期注入是否真的比 patch-wise 后处理更能把 metric 修正传播到无锚区域。

## 当前方法

- Backbone: Marigold / E2E-FT,后续可扩展 Lotus / DepthFM。
- 机制:proxy-anchored CCF + 几何锚采样期烘焙。
- 评测:metric 主表 + affine 对照 + `nfe_real`。
- 锚源:立项先用公开数据集 GT 稀疏采样做 oracle 机制验证;真实锚源后续再替换。

## 四个 Gate

| Gate | 要证明什么 | 判据 |
|---|---|---|
| L0 | CCF 不是单点 x0 或多噪声平均的装饰 | `b_ccf < a_single` 且 `b_ccf < c_multinoise` |
| diag | CCF 不只是降方差,还要抵抗 Tweedie 有偏代理锚攻击 | `abs_bias(b_ccf) < abs_bias(c_multinoise)` |
| L1 | 采样期注入优于同源 patch-wise 后处理 | `C_sample_geo_local < Bp_postproc_patch_ccf` |
| L2 | 近单步可行 | `nfe_real <= 2` 接近高 NFE |

失败解释:
- L0 失败:方法新意退化为 E2E-FT。
- L1 失败:米制卖点不成立。
- L2 失败:只能降级为多步 guidance。
